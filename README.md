# GDT | Gestão e Direito de Trânsito | Marcos Pereira

Versão estruturada para manutenção simples.

## Arquivo principal de manutenção

Edite preferencialmente:

```text
src/content.js
```

Nele você altera título, subtítulo, links, materiais, eixos editoriais, textos institucionais, convites e aviso de construção.

## Rodar localmente

```bash
npm install
npm run dev
```

## Publicar

Na Cloudflare Pages, use:

- Framework preset: Vite
- Build command: npm run build
- Build output directory: dist
